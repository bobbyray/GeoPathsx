Go to www.hillmap.com and define your trail there as follows:
  • Click Toggle Split Screen to get a full screen view.
  • Zoom to the location for your trail. Search for a Place is helpful.
  • Click Paths and edit your trail.
  • Click Tools (Wrench) > Download GPX
    • Select Export Path As: Route (Track is ok too)
    • Click Export GPX
  • The file hillmap_export.gpx is sent to your download folder.

Go to Step 2) Upload here to save the file so you can access it from your Android phone using the MyTrail app.

Upload the gpx file for you trail that you saved from Hillmap.com as follows:
  • If not already signed in, select Sign In>Facebook and log into Facebook. This provides a unique owner id for your trails.
  • Locate the file (hillmap_export.gpx) on you PC using Choose File.
  • Enter a Geo Path Name for the trail.
  • Select a Geo Path option to Share the trail with others:
    • Public to share with anyone.
    • Private for only you to use the trail.
  • Click Upload to save the file so that it can be accessed by the GeoTrail Android phone app.
You can edit your trails as follows:
  • If not already signed in, select Sign In>Facebook and log into Facebook. This provides a unique owner id for your trails.
  • Select a Geo Path to edit.
  • Optionally locate a different gpx file (hillmap_export.gpx) on you PC using Choose File.
  • Optionally enter a different Geo Path Name for the trail.
  • Optionally select a different Geo Path option to Share the trail with others:
    • Public to share with anyone.
    • Private for only you to use the trail.
  • Click Upload to save the trail so that it can be accessed by the GeoTrail Android phone app.
You can views trails as follows:
  • Select a Geo Path to view.
  • All public trails are in the selection list.
  • If not already signed in, select Sign In>Facebook and log into Facebook in order to see your private trails too.

You can create trails as described in the tabs: 1) Define and 2) Upload. The trails you create are accessible online from this web site as well as from an Android phone. You define if a trail is private only to you or is public for anyone to use.

GeoTrail is an Android app for your phone that shows trails, which you select online. You can save trails offline to your phone to be viewed when the Internet is not accessible.

You can automatically track your geo-location. If you are off the trail, a message is shown along with an optional beep. If you are not using automatic tracking, you can find your current location by the MyLoc button in the app and see if you are on the trail.

The GeoTrail app for your Android phone is no longer available in the Google Play Store.

The Stats tab is used to test the history of recorded stats.

Wigo authentication is used instead of Facebook authenication, although text shown in the UI still refers to Facebook. I have not updated the text because this page is only used for testing.
